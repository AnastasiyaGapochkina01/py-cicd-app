# Приложение
## Описание
Простое Python-приложение, которое поддерживает несколько действий через аргумент командной строки `--action`:
1) test — запускает синтетические тесты самого приложения.
2) check — выполняет healthcheck для приложения.
3) build — собирает приложение со случайной версией.
4) apply — выводит строку применения к указанному окружению, принимает дополнительный аргумент --env

## Запуск
```bash
python3 app.py --action <test|check|build|apply> [--env <environment>]

python3 app.py --action apply --env staging
```
Если используется действие apply, аргумент --env обязателен.
